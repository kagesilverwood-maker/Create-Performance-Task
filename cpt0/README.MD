🎯 How it works (simple explanation)
HTML = your page
JavaScript = sends requests (fetch)
Python (Flask) = does logic and sends results back

👉 Think of it like:

Button → JavaScript → Python → JavaScript → Page update
⭐ Why this is the easiest method
No complicated frameworks
No build tools
Works entirely locally
Very beginner-friendly


INSTRUCTIONS TO Start Python server

In VS Code terminal:

python3.13 app.py

You should see:

Running on http://127.0.0.1:5000/
